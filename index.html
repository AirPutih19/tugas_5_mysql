Apa pengertian dari character set?

character set : kumpulan dari karakter yang dapat dipahami oleh komputer

Apa istilah lain dari database?

schema merupakan istilah lain dari database

Tuliskan query untuk menampilkan character set di MYSQL!

SHOW CHARACTER SET;

Tuliskan query untuk melihat collation yang terdapat di MYSQL!

SHOW COLLATION

Sebutkan 5 tipe data numeric pada MYSQL!

INT, TINYINT, SMALLINT, FLOAT, DOUBLE

Sebutkan 5 tipe data string pada MYSQL!

CHAR, VARCHAR, TINYTEXT, TEXT, MEDIUMTEXT

Sebutkan 3 tipe data date pada MYSQL!

DATE, TIME, DATETIME
